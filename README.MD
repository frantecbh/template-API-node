
# Template Start API Node.js

Temmplate de inicialização de um projeto com NodeJS.

## Bibliotecas

npm init - y

npm i typescript @types/node tsx tsup -D

npx tsc --init

npm i fastify

npm i dotenv

npm i zod


## Instalação

Faça o clone do projeto

```bash
  npm install 
  cd my-project
```
